# This is meant to be used with the keyword REVEAL_TITLE_SLIDE_NOTES
# of org-re-reveal.

- To toggle these notes, press ~v~
  - If a slide contains audio, notes might show transcript
- Press ~?~ for key bindings (in particular, ~o~, ~n~, ~p~, ~Ctrl-Shift-f~)
- Presentations support two different PDF formats, see [[https://oer.gitlab.io/hints.html][usage notes]]
  - Concise PDF format (replace ~.html~ and whatever follows in
    [[https://en.wikipedia.org/wiki/Address_bar][address bar]] with ~.pdf~)
  - Print browser view to PDF (add ~?print-pdf~ after ~.html~, then
    print to PDF;
    [[https://revealjs.com/pdf-export/][suggested settings]])
- If you find the amount of outgoing links to be distracting, see
  [[https://oer.gitlab.io/hints.html][usage notes]]
  - Add ~?hidelinks~ (maybe with a number) after ~.html~
- See [[https://oer.gitlab.io/hints.html][usage notes]] for other non-obvious features
